# Система ачивок для SpaceAttack

## 📋 Обзор

Предлагаемая система достижений из **40+ ачивок**, разбитых на категории. Ачивки — мощный инструмент удержания, дающий игрокам цели за пределами основного геймплея и стимулирующий повторные запуски.

**Типы ачивок:**
- 🟢 **Бронзовые** (легко получить) — знакомство с механиками
- 🔵 **Серебряные** (средняя сложность) — углублённое освоение
- 🟣 **Золотые** (сложно) — мастерство и долгая игра
- 🔴 **Легендарные** (крайне сложно) — вызов для хардкорных игроков

## 🛠️ Техническая реализация

### Новая структура данных в SaveManager:

```gdscript
var achievements: Dictionary = {
	"first_blood": {"unlocked": false, "progress": 0},
	"veteran": {"unlocked": false, "progress": 0},
	# ... все ачивки
}
var total_achievements: int = 0
var achievements_tracked: Dictionary = {
	"enemies_killed": 0,
	"bosses_killed": 0,
	"highest_wave": 0,
	"single_game_score": 0,
	"goliath_charge_kills": 0,
	"phantom_dashes": 0,
	"nanobots_healed": 0,
	"chests_opened": 0,
	"shockwave_used": 0,
	"rocket_kills": 0,
	"forsage_procs": 0,
	"health_packs_collected": 0,
	"cocoon_shield_blocked": 0,
}
```

### Необходимые изменения:

1. **SaveManager.gd** — добавить словарь `achievements`, методы `check_achievement()`, `unlock_achievement()`, `get_achievement_progress()`
2. **Main.gd** — добавить триггеры ачивок в `add_score()`, `_on_enemy_killed()`, `game_over()`, `end_boss_fight()`
3. **Player.gd** — добавить триггеры ачивок в `take_damage()`, `die()`, `_do_dash()`, `_do_goliath_charge()`, `try_activate_shockwave()`
4. **Hangar.gd** — добавить экран ачивок (кнопка Rewards), отображение прогресса
5. **Новый UI** — окно ачивок с категориями и прогресс-барами
6. **Обновление сохранения** — версионирование сейвов (при загрузке без ачивок — инициализировать пустым словарём)

### Эффект удержания:

- **Короткая сессия (1-5 мин)**: ачивки за первую кровь, сбор монет, открытие сундуков
- **Средняя сессия (10-20 мин)**: прохождение волн, убийство боссов, использование оружия
- **Долгая сессия (30+ мин)**: полное прохождение, высокие счета, экстремальные челленджи
- **Реиграбельность**: ачивки на разные корабли, разное оружие, стили игры
- **Социальный фактор**: общий прогресс (25%/50%/75%/100%) мотивирует вернуться
